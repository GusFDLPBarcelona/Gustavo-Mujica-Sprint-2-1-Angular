# Sprint 2.1 — JavaScript Fundamentals

Colección de ejercicios prácticos de JavaScript moderno organizados por tema y nivel de dificultad (Nivel 1 → 3). Cada ejercicio se ejecuta en el navegador e incluye interacción con el DOM y visualización de resultados en la consola del navegador.

## Demo

[Ver demo en vivo](https://gusfdlpbarcelona.github.io/Gustavo-Mujica-Sprint-2-1-Angular/)

## Temas cubiertos

| # | Tema | Conceptos clave |
|---|------|----------------|
| 1 | Arrow Functions | Sintaxis compacta, `this` léxico, `setTimeout` |
| 2 | Operador Ternario | Expresiones condicionales en una línea |
| 3 | Callbacks | Funciones de orden superior, flujo asíncrono |
| 4 | Rest & Spread | `...rest` en parámetros, `...spread` en arrays |
| 5 | Array Transformations | `map`, `filter`, `reduce`, `every`, `some` |
| 6 | Array Loops | `for`, `forEach`, `for...of`, `for...in`, `entries()` |
| 7 | Promises & Async/Await | `new Promise`, `.then`, `.catch`, `async/await`, `Promise.all` |

## Estructura

Cada tema contiene tres niveles de dificultad:

```
B1-X-NombreTema/
├── Nivel-1/   # Conceptos básicos
├── Nivel-2/   # Uso intermedio
└── Nivel-3/   # Casos más avanzados
```

Dentro de cada nivel: un `index.html` con la interfaz y uno o varios archivos `.js` con la lógica.

## Cómo usar

1. Abre el `index.html` del ejercicio que quieras revisar directamente en el navegador
2. Abre la consola del navegador (F12 → Console)
3. Interactúa con los botones o inputs de la página
4. Observa los resultados en la consola

## Qué demuestra este proyecto

- Dominio de **ES6+**: arrow functions, destructuring, template literals, spread/rest
- Manejo de **asincronía**: callbacks, Promises (resolve/reject/.then/.catch), async/await, Promise.all
- Manipulación del **DOM**: eventos, `getElementById`, actualización dinámica de contenido
- **Métodos funcionales de array**: map, filter, reduce, every, some, entries
